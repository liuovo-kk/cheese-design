# Upload 上传

极其优雅、支持拖拽与照片墙的企业级数据录入组件，为用户提供极其顺滑的文件上传体验。

## 💡 何时使用

- 当你需要将本地的图片、文档、视频等文件上传至服务器时。
- 当你需要大厂级别的交互体验（如拖拽上传、精准进度条、大图全屏预览）时。
- 当你需要在一个表单中收集用户的多媒体资料时。
